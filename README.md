## About

Cat Facts is a simple yet engaging extension that fetches random cat facts from the [MeowFacts API](https://meowfacts.herokuapp.com/). Perfect for cat lovers who want to learn something new about their feline friends or just need a quick dose of adorable information.

## Usage

1. Open Raycast and type "Cat Facts"
2. Press `Enter` to fetch random cat facts
3. Use `Cmd+F` to star/unstar facts as favorites
4. Press `Cmd+T` to toggle between facts and favorites
5. Copy individual facts or all facts to clipboard

## Preferences

- **Fact Count**: Choose how many facts to fetch (1-50)
- **Language**: Select your preferred language for facts

## Keyboard Shortcuts

- `Cmd+F` - Toggle favorite status
- `Cmd+T` - Switch between facts and favorites
- `Cmd+Shift+Delete` - Clear all favorites

## Installation

This extension is part of a Raycast technical assessment. To use it:

1. Clone this repository

3. Run the extension locally
